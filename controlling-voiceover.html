Controlling VoiceOver

Basic controls

To navigate interactive parts of a website (like buttons and forms) you can use the keyboard. You need to use these VoiceOver commands to help you access all the content.

The default VoiceOver (VO) button is ctrl + alt. So when it says VO + P in the table below, that means press ctrl, alt and p together.

cmd + F5 enable and disable VoiceOver
VO + A start reading
Ctrl stop reading
VO + Shift + ↓ go into a section
VO + Shift + ↑ exit a section
VO + → next
VO + ← previous
Enter click a link or button
Space select a checkbox, radio or dropdown

The rotor

The rotor lets you navigate by element type, for example headings and links.

VO + U open the rotor
← and → switch between element types
↑ and ↓ select an element
Space or Enter go to selected element
