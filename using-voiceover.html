Using VoiceOver

Voice over is the built in screen reader that all modern Mac computers, iPhones and iPads have.

It can be tricky to use, but this shouldn't stop you using them to test the accessibility of a web page. It will also help you get a feel for how difficult it can be, especially for someone who might have recently lost their sight. Even more reason to help make everything as accessible as possible.
